# 80s Arcade Tetris

브라우저에서 실행되는 단일 HTML 파일 테트리스 게임. 외부 라이브러리 없이 순수 Canvas API와 JavaScript로 구현됐다.

## 구조

```
tetris80s.html
├── Starfield (bgCanvas)   — 배경 별빛 애니메이션
├── Game Canvas (gameCanvas) — 메인 게임 보드 (10×20)
├── Next Canvas (nextCanvas) — 다음 피스 미리보기
└── Side Panel             — 점수 / 레벨 / 라인 수 표시
```

## 게임 로직

**피스 생성 — 7-Bag 랜덤화**
7종류 테트로미노(I, O, T, S, Z, J, L)를 무작위로 섞은 뒤 순서대로 꺼낸다. 한 묶음이 소진되면 다시 섞어 보충하므로, 같은 피스가 지나치게 몰리지 않는다.

**낙하 루프**
`requestAnimationFrame`으로 매 프레임마다 타임스탬프를 확인한다. 마지막 낙하 이후 `dropMs` 이상 경과하면 피스를 한 칸 내린다. 레벨이 오를수록 `dropMs`가 줄어든다(최소 80ms).

**충돌 감지**
`valid(shape, x, y)` 함수가 피스의 각 셀을 순회해 보드 경계 및 기존 블록과의 겹침을 확인한다. 이동·회전·낙하 전에 항상 호출된다.

**회전 — 월킥**
회전 후 `valid`가 실패하면 좌우로 최대 2칸씩 밀어 재시도한다. 벽이나 블록에 붙어있어도 회전이 가능한 이유다.

**라인 클리어**
피스가 고정될 때 모든 행을 검사해 가득 찬 행을 기록한다. 180ms 동안 흰색 플래시를 표시한 뒤, 해당 행을 제거하고 위 행들을 한 칸씩 내린다.

**점수**
| 라인 | 기본 점수 | 비고 |
|------|-----------|------|
| 1    | 100 × 레벨 | |
| 2    | 300 × 레벨 | |
| 3    | 500 × 레벨 | |
| 4    | 800 × 레벨 | 테트리스 |
| 소프트 드롭 | +1/칸 | |
| 하드 드롭 | +2/칸 | |

10라인마다 레벨이 1 오른다.

## 렌더링

- **블록** — 색상 채우기 → 상단/좌측 하이라이트 → 하단/우측 그림자 → `shadowBlur` 글로우 순으로 그려 3D 입체감과 네온 발광을 동시에 표현한다.
- **고스트 피스** — 현재 피스의 최종 착지 위치를 반투명(18%)으로 미리 표시한다.
- **CRT 효과** — CSS `repeating-linear-gradient`로 스캔라인을, `radial-gradient`로 비네팅을 구현한다. 캔버스 자체를 수정하지 않으므로 렌더링 성능에 영향이 없다.

## 조작

| 키 | 동작 |
|----|------|
| `← →` | 좌우 이동 |
| `↑` | 회전 (월킥 포함) |
| `↓` | 소프트 드롭 |
| `Space` | 하드 드롭 |
| `P` | 일시정지 / 재개 |
| `Enter` | 게임 오버 후 재시작 |

모바일에서는 화면 하단의 터치 버튼을 사용한다.
